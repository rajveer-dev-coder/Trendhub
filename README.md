# Trendhub
# 🛒 Trendhub.co — Premium E-Commerce Interface

![Trendhub Preview](https://img.shields.io/badge/UI%2FUX-Premium-blue) ![Version](https://img.shields.io/badge/Version-1.0.0-green) ![License](https://img.shields.io/badge/License-MIT-yellow)

**Trendhub.co** ek high-end, modern e-commerce web application hai jise pure **HTML5, CSS3 (Modern Grid/Flex), aur Vanilla JavaScript** ka use karke banaya gaya hai. Iska design aesthetic "Senior Developer" standards ko follow karta hai, jisme clean typography, intuitive navigation, aur smooth micro-interactions par focus kiya gaya hai.

---

## ✨ Key Features

* **⚡ High-Performance UI:** Bina kisi heavy framework ke fast loading aur smooth transitions.
* **📱 Fully Responsive:** Mobile, Tablet, aur Desktop screens ke liye perfectly optimized layout.
* **🛒 Interactive Cart System:** JavaScript-driven cart logic jo real-time mein count update karta hai.
* **🎨 Premium Aesthetic:** Soft shadows, modern color palette, aur Lucide icons ka integration.
* **📦 Dynamic Rendering:** Product data ko arrays se render kiya gaya hai, jise easily backend API se connect kiya ja sakta hai.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables, Grid Layout)
* **Logic:** Vanilla JavaScript (ES6+)
* **Icons:** [Lucide Icons](https://lucide.dev/)
* **Fonts:** Inter (via Google Fonts)

## 🚀 Getting Started

1.  Is repository ko clone karein:
    ```bash
    git clone [https://github.com/your-username/trendhub-ecommerce.git](https://github.com/your-username/trendhub-ecommerce.git)
    ```
2.  `index.html` file ko kisi bhi modern browser mein open karein.
3.  That's it! Kisi bhi server setup ki zaroorat nahi hai (Frontend-only demo).
